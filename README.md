# Blogify Backend

Backend services for the **Blogify platform** built using **Java and Spring Boot** with a **microservices architecture**.

The backend is divided into separate services to handle different responsibilities such as blog management and user management.

---

## Features

* User management service
* Blog management service
* RESTful API endpoints
* Microservices architecture
* Scalable backend design

---

## Tech Stack

* Java
* Spring Boot
* Maven
* REST API
* Microservices Architecture

---

## Project Structure

blogify-backend
│
├── blog-service
├── user-service
└── .gitignore

---

## Getting Started

### Clone the repository

git clone https://github.com/Rishijha121003/blogify-backend.git

### Navigate into the project

cd blogify-backend

### Run the services

Each service can be started independently.

Example:

cd blog-service
mvn spring-boot:run

or

cd user-service
mvn spring-boot:run
